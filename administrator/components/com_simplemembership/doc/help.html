Simple Membership

For start use Simple Membership you need:

  1. 1. Install Simple Membership component.
  2. 2. Install Simple Membership module.
  3. 3. Goto to Simple Membership component.
  4. 4. Go to 'Groups'.
    1. 4.1 Create Simple Membership group and connect this to joomla groups.
    2. 4.2 Set time to expire for this group, after which user will be disapproved and belong to default joomla group(6.2).
    3. 4.3 Choose connected virtuemart product (it's not avialable in free version), after purchasing which user will be approved to this group for next time(4.2).
    4. 4.4 Type post-register notes for this group, it will show after registration.
  5. 5. Go to 'Settings Frontend'.
    1. 5.1 Choose Add user to send email : 'YES' or 'NO'.
    2. 5.2 Type pre-register message, which will show before registration.
    3. 5.3 Type activation message, which will show when activation.
  6. 6. Go to 'Settings Backend'.
    1. 6.1Type "Add user" email, at this mail will send 'Administrator notification message on account creation' (if it turned on 5.1).
    2. 6.2 Select 'default joomla group', there will be disapproved users from Simple Membership groups after expired time (4.2).
    3. 6.3 Type 'Administrator notification message on account creation', this will send at email(6.1).
    4. 6.4 Type ' Notification message on account creation', this message will send to user, who registering.
    5. 6.5 Type 'Notification message on account enabled', this message will send to user, who enabled.
    6. 6.6 Type 'Notification message on account approved.', this message will send to user, who approved to Simple Membership group.
    7. 6.7 Type 'Notification message on account disapproved', this message will send to user, who disapproved to Simple Membership group.
  7. 7. Go to 'Users', there you can: add new user, delete user, edit user, approve and disapprove users, enabled and disabled users for joomla, synchronize and check for expiring Simple Membership users with joomla users.
    1. 7.1 Adding new user, please click button 'new user' in tool-bar
      1. 7.1.1 Please enter 'name'
      2. 7.1.2 Please enter 'username'
      3. 7.1.3 Please select 'MemberGroup', or select joomla group(in this case user will not be in simple membership group)
      4. 7.1.3 Please enter 'Password'
      5. 7.1.4 Please 'Verify Password'
      6. 7.1.5 Please if you need fill out fields in field-set ' Basic Settings' (it refers to system user setting:Backend Template Style, Backend language, Frontend language, Editor, Help Site, Time zone)
      7. 7.1.6 Please if you need fill out fields in field-set 'User Profile' (but it possible only if in your system enabled plug-in 'user-profile' or 'user-simplemembership')
    2. 7.2 Deleting user, please click button 'delete' in tool-bar, user will be deleted from simple membership and joomla
    3. 7.3 Editing user, click user link in column 'name'
      1. 7.3.1 You can change 'name'
      2. 7.3.2 You can change 'username'
      3. 7.3.3 You can change 'MemberGroup', or select joomla group(in this case user will not be in simple membership group)
      4. 7.3.3 You can change 'Password'
      5. 7.3.4 You can change 'Verify Password'
      6. 7.3.5 You can change if you need fill out fields in field-set ' Basic Settings' (it refers to system user setting:Backend Template Style, Backend language, Frontend language, Editor, Help Site, Time zone)
      7. 7.3.6 You can change if you need fill out fields in field-set 'User Profile' (but it possible only if in your system enabled plug-in 'user-profile' or 'user-simplemembership')
    4. 7.4 Synchronize and checking user, click button 'Sync users'
    5. Synchronizing needed for synchronize joomla users and Simple Membership users, and for checking expiring users for Membership group, it's recommended to do this, that's because only users who logins to site has automatic synchronize and checking , another user if they don't login to site don't have.
    6. 7.5 Approve and disapprove users. You can approve and disapprove users, you need click tick in column 'approved'
    7. 7.6 Enable and disable users for joomla. That's similar 7.5
